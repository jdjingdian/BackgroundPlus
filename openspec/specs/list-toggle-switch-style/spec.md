## ADDED Requirements

### Requirement: 列表行启用状态必须用开关样式呈现
系统 SHALL 用 switch 风格的 toggle 表示每一行的启用状态，默认视觉必须是 macOS 风格的滑动开关，且控件区域必须提供独立点击热区，以避免与行内其他操作（例如 `→`）冲突。

#### 场景: 启用某条目
- **当** 用户在某行点击 switch 控件并将状态变为“启用”
- **那么** 该条目的 `isEnabled` 状态必须更新为 `true`，切换控件必须以 switch 轨迹的动画反馈状态变更，并继续暴露辅助标识（如 `accessibilityLabel`）告知当前状态

#### 场景: 禁用某条目
- **当** 用户在某行点击 switch 控件并将状态变为“禁用”
- **那么** 该条目的 `isEnabled` 状态必须更新为 `false`，Switch 必须清晰展示关闭状态，并在视觉与辅助功能上区分于启用状态
